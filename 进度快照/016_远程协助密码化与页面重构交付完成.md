# 进度快照 016 - 远程协助密码化与页面重构交付完成

## 本次已完成

1. 完成“本机状态”模块迁移
   - 从远程协助页移除本机状态卡片。
   - 在链接状态页顶部新增默认折叠的“本机状态”面板。
   - 展开后继续显示安装状态、服务状态、监听状态、本机虚拟 IP、版本、问题列表与修复按钮。

2. 完成远程协助访问密码能力
   - 远程协助页顶部新增“设置远程密码”按钮，位置在“刷新”左边。
   - 支持设置非空密码后无人值守接入。
   - 支持清空密码后恢复为手动点击接受。
   - 主项目新增 `configureAccessPassword` 调用链。
   - `vntcrustdesk` 新增 `--configure-access-password <password>` 无界面配置入口。

3. 完成密码复杂度前端限制移除
   - `vntcrustdesk` 桌面端“Set Password”弹窗去掉 8 位/大小写/数字强校验。
   - `vntcrustdesk` 移动端永久密码弹窗去掉最短长度前端校验。
   - 仍保留底层加密实现本身的技术上限。

4. 完成远程协助在线用户过滤
   - 远程协助页只显示 `isOnline == true` 的用户。
   - 无在线目标时显示空状态提示。
   - 点击在线用户卡片仅回填虚拟 IP，不改访问密码输入框。

5. 完成目标连接双输入框改造
   - 原单输入框拆分为“虚拟IP”和“访问密码”。
   - 连接时支持仅 IP 发起普通连接。
   - 连接时支持 IP + 密码透传到 `vntcrustdesk --connect ... --password ...`。

6. 完成完整源码构建与主项目制品接入
